# Eden Nursery - Premium Botanical Boutique

Experience the cinematic world of premium indoor plants. This project is a high-performance, visually immersive e-commerce platform built with Next.js 15, Three.js, and Supabase.

## Features
- **Cinematic Experience**: Immersive 3D visuals and smooth animations.
- **Product Collection**: Curated selection of plants with detailed care instructions.
- **Dynamic Cart**: Real-time cart management using Zustand.
- **Admin Dashboard**: Secure panel for managing inventory, products, and categories.

## Admin Access
The admin dashboard is located at `/admin`. It provides:
- **Inventory Management**: View and update stock levels.
- **Product CRUD**: Add, edit, and delete botanical species.
- **Category Management**: Organize the collection.
- **Real-time Stats**: Insights into stock levels and catalog size.

## Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS 4
- **Animations**: GSAP, Framer Motion
- **3D**: Three.js, @react-three/fiber, @react-three/drei
- **Backend**: Supabase
- **State Management**: Zustand
